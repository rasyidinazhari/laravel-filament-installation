Bagian G

Instalasi Filament

3 command untuk setup Filament di project Laravel.

Terminal — Step 1
# Masuk ke folder project cd akademik # Install Filament package composer require filament/filament
Terminal — Step 2
# Install panel admin php artisan filament:install --panels
Terminal — Step 3
# Jalankan migrasi php artisan migrate
Bagian H & I

Jalankan & Buat Admin

🚀 Menjalankan Aplikasi

Terminal
php artisan serve # Buka browser: http://127.0.0.1:8000/admin

👤 Buat User Admin

Terminal
php artisan make:filament-user # Isi data berikut: Name: Administrator Email: admin@kampus.ac.id Password: admin12345
💡 Login ke http://127.0.0.1:8000/admin menggunakan email & password yang sudah dibuat.
Bagian J

Membatasi Akses Admin Only

Implementasi FilamentUser interface pada model User.

// app/Models/User.php use Filament\Models\Contracts\FilamentUser; use Filament\Panel; class User extends Authenticatable implements FilamentUser { public function canAccessPanel(Panel $panel): bool { return $this->role === 'admin'; } }

Resource Kelas Matkul

Terminal
php artisan make:filament-resource KelasMatkul
Form — Select Relasi

Table — Relasi Column
